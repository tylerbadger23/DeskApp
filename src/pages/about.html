Hello!

My name is Tyler Laceby. Im a Computer Science Student developing this app in my free time. Its currently open to the public for my friends, family, and possible employers to see and use. I can assure there will be some minor bugs and hickups while using but please feel free to leave feedback if you encounter one you would like to have fixed. Hope you enjoy and like what you see.


This application is currently written in Electron.js and is hosted on a Node web server. Since Electron powers the desktop support i can have access to more advanced features and personality for users that can be used offline. I am working on writing the API for access to this information between the users desktop device and browser with sharing and syncying of data. Currently offline use is buggy and will cause a major crash.


This project is primarily developed as a side hobby and has not been tested besides on my MacBook and Destop machine.
